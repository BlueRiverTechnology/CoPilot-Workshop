## Appendix: GitHub Copilot Features Covered

### Core Copilot Modes
| Mode | Purpose | When to Use |
|------|---------|-------------|
| **Ask Mode** | Q&A, explanations | Understanding code, getting help |
| **Edit Mode** | Controlled edits to working set | Precise multi-file changes |
| **Agent Mode** | Autonomous implementation | Complex features, multi-step tasks |
| **Plan Mode** | Create implementation plans | Before major features |

### Context Variables (#mentions)
| Variable | Purpose |
|----------|---------|
| `#file` | Reference specific file |
| `#folder` | Reference directory |
| `#codebase` | Search entire codebase |
| `#problems` | Include IDE errors/warnings |
| `#terminalSelection` | Include terminal output |
| `#fetch` | Fetch web content |

### Copilot Customization
| Feature | File | Purpose |
|---------|------|---------|
| **Custom Instructions** | `.github/copilot-instructions.md` | Project-wide AI behavior rules |
| **Prompt Files** | `.prompt.md` | Reusable prompt templates |
| **Custom Agents** | `.agent.md` | Specialized AI personas |
| **Ignore Files** | `.copilotignore` | Exclude sensitive files from AI |

### Advanced Features
| Feature | Description |
|---------|-------------|
| **Subagents** | Delegate tasks to isolated agents for parallel work |
| **Code Review** | AI-powered PR reviews on GitHub.com |
| **Handoffs** | Transition between agents with context |
| **MCP Servers** | External tool integration |
