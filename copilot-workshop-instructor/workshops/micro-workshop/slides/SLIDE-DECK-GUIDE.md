- Session 1: Blue (Shield icon)

## Icon Suggestions
Use consistent icon set (Lucide, Heroicons, or Font Awesome):
- Shield for security
- Brain for thinking/planning
- Rocket for building/speed
- Trophy for achievements
- Robot for AI/agents
- File/folder for context
- Lock for security features
- Checkmark for success
- Timer for exercises

## Typography
- **Headings:** Bold, clean sans-serif (e.g., Inter, SF Pro)
- **Body:** Regular weight, good line height (16-18px)
- **Code:** Monospace font (JetBrains Mono, Fira Code)
- **Slide numbers:** Small, unobtrusive bottom corner

## During Workshop

## Critical Teaching Points
1. **Rules are living documents** - iterate based on what works
2. **Checkpoints are safety nets** - git reset is always available
3. **TDD gives confidence** - AI + tests = verified correctness
4. **Custom Agents = HOW, Memory Bank = WHERE** - key distinction
5. **70/30 is critical** - AI speed + human quality = professional results
